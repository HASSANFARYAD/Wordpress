Autoregression

Autoregression is a time series model that uses observations
from previous time steps as input to a regression equation to predict the value at
the next time step. The autoregressive model specifies that the output variable
depends linearly on its own previous values. In this technique input variables are
taken as observations at previous time steps, called lag variables.
